## Hướng dẫn Import File .json vào Postman

### 1. Mở ứng dụng Postman
- Khởi chạy ứng dụng **Postman** trên máy tính.

---

### 2. Chọn chức năng Import
- Ở góc trên bên trái, nhấn vào nút **Import**.
- Một cửa sổ Import sẽ xuất hiện.

---

### 3. Chọn file JSON cần import
- Trong cửa sổ Import, chọn tab **File**.
- Nhấn **Upload Files** hoặc kéo-thả file `.json` vào khu vực chọn file.
- Duyệt chọn file cần import (ví dụ: `RecruitmentAPI.postman_collection.json`).

---

### 4. Xác nhận Import
- Postman hiển thị thông tin preview của file trước khi import.
- Nhấn **Import** để hoàn tất.

---

### Kết quả
- Postman sẽ tự động tạo:
  - **Collection** (nếu file là Postman Collection `.json`)
- Bạn có thể thấy nội dung vừa được import ở sidebar bên trái.
